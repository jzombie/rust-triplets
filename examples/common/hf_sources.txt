# Rules:
# - comments start with '#'
# - at least one of anchor= positive= context= text= is required per line
# - accepted keys: anchor= positive= context= text= trust= source_id=
# - context= and text= accept comma-delimited columns
# - anchor= positive= text= coalesce: first non-empty candidate column is used
# - trust= (optional) overrides the default trust score (0.5); value in [0.0, 1.0]
# - source_id= (optional) overrides the auto-derived slug for this source
# - ClassLabel columns (integer-encoded categories) are resolved to their string
#   names automatically via the datasets-server /info endpoint at load time;
#   no manual annotation is needed
